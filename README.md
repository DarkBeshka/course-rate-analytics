# Аналитика качества образовательных программ в онлайн-школе

## О проекте

Разработан аналитический дашборд для мониторинга качества обучения в онлайн-школе программирования. Решение позволяет агрегировать обратную связь от преподавателей, отслеживать изменения удовлетворенности обучением и выявлять направления для улучшения образовательного продукта.

Проект выполнен в роли аналитика-методолога и ориентирован на поддержку управленческих решений в обучении.

## Бизнес-задача

Школа ежемесячно получает сотни отзывов от преподавателей по десяткам курсов. Однако по ним не проводилась аналитика.

Необходимо было создать единый инструмент, который позволяет:

- контролировать качество обучения;
- отслеживать динамику удовлетворенности;
- выявлять проблемные курсы;
- анализировать качество методических материалов;
- оценивать устойчивость образовательных результатов.

## Использованные инструменты
- Power BI
- Power Query
- DAX
- Google Таблицы и Google Формы

## Подготовка данных

В рамках проекта были выполнены:

1. очистка и стандартизация данных отзывов;
2. создание календарной модели;
3. расчет агрегированных показателей;
4. разработка показателя взвешенной оценки;
5. настройка интерактивной фильтрации по курсам.

## Разработанные метрики
**Средняя оценка курса**. Базовый показатель удовлетворенности обучением.

**Взвешенная оценка**. Бизнес-приоритетными отзывами считались отзывы от опытных преподавателей. Поэтому отзывам преподавателей был присвоен вес в зависимости от квартиля по стажу от 0.25 до 1(чем больше опыта, тем больше вес). Также в взвешенной оценке был учтен нежелательный случай наличия выявленных ошибок в курсах, но при этом завышенной оценки. Итоговая взвешенная оценка формируется по формуле:

$5-(5-"Оценка\_преподавателя")*"Вес\_голоса"-ЕСЛИ("Найдены\_ошибки\_в\_ММ"; 1; 0)*"Вес\_голоса"$

**Количество отзывов**. Позволяет оценивать статистическую надежность результатов.

**Стандартное отклонение**. Используется для оценки стабильности качества обучения.

**Анализ проблем методических материалов**. Позволяет отслеживать частоту замечаний по содержанию и структуре курсов.

## Реализованные аналитические блоки
1. Мониторинг качества
- динамика средней оценки;
- динамика взвешенной оценки;
- средние показатели по школе.
2. Анализ продуктового портфеля
- распределение отзывов по курсам;
- выявление наиболее востребованных направлений обучения.
3. Анализ преподавателей
- рейтинг преподавателей по объему обратной связи;
- возможность дальнейшего расширения до оценки качества преподавания.
4. Анализ методических материалов
- классификация замечаний;
- мониторинг качества контента.

## Основные выводы

По результатам анализа более 35 тысяч отзывов:

- средняя оценка обучения составила 4,94 балла;
- средняя взвешенная оценка составила 4,92 балла;
- качество обучения в среднем оставалось стабильным на протяжении года, однако были выявлены курсы, имеющие резкие спады в средней оценке в связи с осложнением доступа к нужным ресурсам;
- более 97% преподавателей не выявили проблем в методических материалах;
- наиболее востребованными направлениями стали Roblox Lua, Python и Unity;
- существенных признаков деградации качества образовательного продукта не обнаружено.

## Практическая ценность

Дашборд позволил перевести оценку качества обучения из ручного анализа отдельных отзывов в регулярную систему мониторинга на основе данных.

Результаты *ежемесячно* используются для:

- принятия решений по развитию курсов;
- распределения обязанностей сотрудников отдела обучения;
- контроля качества образовательного продукта;
- оценки эффективности изменений в программах обучения;
- выявления потенциальных зон риска.
